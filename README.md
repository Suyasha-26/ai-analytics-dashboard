# AI Powered Analytics Dashboard

An interactive AI-powered analytics dashboard built using Streamlit, Pandas, Plotly, and Gemini AI. The application allows users to upload datasets, perform dynamic filtering, generate visualizations, analyze correlations, and receive AI-generated business insights.

---

## Features

- CSV File Upload
- Dynamic Data Filtering
- Interactive Visualizations
- Correlation Heatmap
- AI Generated Insights
- Chat With Your Data
- Download Filtered Dataset
- Multiple Chart Types

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Gemini AI
- dotenv

---

## Installation

1. Clone the repository

```bash
git clone <your_repository_link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create a `.env` file and add your Gemini API key

```env
GEMINI_API_KEY=your_api_key
```

4. Run the application

```bash
streamlit run app.py
```
